Installing

Installing is too easy. Run setup program and follow it.
Remember: When you install any add-on to Outlook need quit this program.

Using

After install and start Outlook, the system add new tab in Ribbon with name "Add-Ins". There is new section "Google Contact". There are three buttons: Configuration, Synchronize and About.

Configuration

Before first using you must configure Google account and synchronization way. Group "First synchronize" is for new synchronization or for re-connect synchronization between systems. After first synchronization ends the checkbox “Is First synchronize “ uncheck and in every next synchronization is use settings from "Standard synchronize setup".
There is possible set logging level and logging directory.
 

In Advanced settings is possible set more options. First one is configuration for Cache system. This mean use local cache for contacts and its images. There are two buttons witch delete cached data for Outlook or Google.

Parameter Time To Live mean – how many days are cache data valid from last synchronization. After this time are all data read from source.

Next available options on Advanced Configuration is Clear synchronization keys in source systems.

Synchronization

System synchronizes your Outlook contacts with Google Contacts by your request. After synchronization is show window with information about changes in both systems.

In fist time and after more than one hour inactivity need authorize application for Google Contact API. Program open default internet browser and request your credentials. This is no more write to  application.

After login to Google account (if you don’t login automatically), you must approve application permissions.

When press “Allow access” system generate access code for application. This code is valid only short time. You copy this code and insert to application window.

Insert code to this windows and after press OK system start synchronize contact.
